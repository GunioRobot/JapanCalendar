Class JapanCalendar
In: lib/japan_calendar.rb
Parent: Object

JapanCalendarクラスの機能

  1. その日が、平日かどうか取得できます。
  2. その日が、祝祭日かどうか取得できます。
  3. 祝祭日のとき、日本語名を取得できます。
  4. 土曜日を休日扱いにするかしないか設定できます。
    • 初期設定では土曜日は平日扱い
